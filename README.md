# Keefer Law Firm Website

A responsive law firm website rebuilt from a reference design.

## Setup

1. Clone or download the repository
2. Open `index.html` (vanilla CSS) or `index-tailwind.html` (Tailwind CSS) in a browser
3. No build process required

## Tech Stack

| Version | Technology |
|---------|------------|
| `index.html` | HTML5, Vanilla CSS |
| `index-tailwind.html` | HTML5, Tailwind CSS (CDN) |

## Features

- **Responsive Design** - Desktop, tablet, and mobile layouts
- **Header** - Logo and contact phone number
- **Navigation** - Red gradient menu bar with hover effects
- **Hero Section** - Banner image with contact form
- **Info Boxes** - DUI info, Law Blog, Case Results, Credentials
- **Practice Areas** - Grey sidebar with red accent strips
- **Right Sidebar** - Testimonials, Reports, Contact, Social links
- **Mobile Friendly** - Stacked layouts on smaller screens
